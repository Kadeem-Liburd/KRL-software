Friendly Dorm Management System

Friendly Dorm Management System was a group project that I was a part of.

This project focus was to make a website where students at Centennial College would

be able to register for a Dorm room for the semester.The system capablities are,

display information to everyone, registration process, dorm room status, and able

to send anonymous complaints.



The Systems Benefits are:

-Maintain student records

-Data can be accessed easily by authority

-System reduce human efforts

-Residence can use online service to register and pay bills

-System's user friendly features will attract more student towards dorm amenities



The role Kadeem play in this project was to be the

developers. Kadeem also did the tasks for Admin & Front desk users.

Lastly, have to connect the programs and document the functionalities.
